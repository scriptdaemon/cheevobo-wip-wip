dice(bot) → {Object}

Simulate a dice roll. Responds with the result of each die roll as a single, space-delimited string. Accepts one (1) optional argument: * roll (default: '1d6') - Expects a string in #d# format; the first # is the number of dice to - roll and the second # is the max of each die. This value is coerced to a - single, space-delimited string of the result rolls of each die.
